## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| provider UIs → workflow run-time | Trusted publishers, environment reviewers, and secrets configured outside git authorize the run |
| dispatch dry-run → external indexes | The dry-run uploads to TestPyPI (never real PyPI) |
