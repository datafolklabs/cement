# Phase 05.4 Plan 03: release.yml Dry-Run Path Summary

Created the dry-run-reachable half of `.github/workflows/release.yml` — a strict-semver-tag / workflow_dispatch pipeline running preflight → shared gates → isolated build → TestPyPI publish + 5-Python install smoke → workflow_dispatch dry-run terminus, stopping before any real-PyPI publish.

## What Was Built

**Task 1 — scaffold (commit 71718dca):**
- `on: push: tags: ['[0-9]+.[0-9]+.[0-9]+']` (excludes moving tags `3`/`3.0`, Pitfall 7) plus `workflow_dispatch` (dry-run from any branch, D-06).
- Top-level `permissions: contents: read` (V14 least privilege); `concurrency` group keyed on workflow+ref with `cancel-in-progress: false`.
- `preflight` job (checkout@v7.0.0, setup-python@v6.3.0, setup-pdm@v4.5, pdm install): on push asserts strict semver + `tag == get_version()`; on both events asserts CHANGELOG has a finalized `## <ver>` section (not `DEVELOPMENT`); exports `version` to `$GITHUB_OUTPUT`.
- `gates` job: `needs: preflight`, `uses: ./.github/workflows/gates.yml`, `with: full-os-matrix: true` — the SAME reusable workflow PR CI calls. **D-13 invariant now holds: `grep -c 'uses: ./.github/workflows/gates.yml' .github/workflows/*.yml` == 2** (build_and_test.yml + release.yml).

**Task 2 — build + TestPyPI stage + dry-run stop (commit b6f1b7e5):**
- `build` (needs: [preflight, gates]): `pdm build`, uploads `dist/` via `upload-artifact@v7.0.1` — isolated from publish so identical bytes flow downstream.
- `testpypi-publish` (needs: build, `environment: testpypi`, `permissions: id-token: write`): downloads the dist artifact via `download-artifact@v8.0.1`, publishes with `pypa/gh-action-pypi-publish@v1.14.0` to `https://test.pypi.org/legacy/` with `skip-existing: true` (D-08 idempotent re-runs).
- `testpypi-smoke` (needs: testpypi-publish): matrix over Python 3.10–3.14 running `python scripts/testpypi-smoke.py ${{ needs.preflight.outputs.version }}` — plain zero-dep install from TestPyPI (Pitfall 2).
- `dry-run-summary` (needs: testpypi-smoke, `if: github.event_name == 'workflow_dispatch'`): writes a `$GITHUB_STEP_SUMMARY` line and STOPS — dispatch never publishes.
