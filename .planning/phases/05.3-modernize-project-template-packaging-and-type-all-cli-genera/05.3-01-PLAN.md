<objective>
Make the generated `project` template pass `make comply` (ruff + mypy) + `make test`
green immediately after `cement generate project -D <dest> --defaults`, with zero manual
fixups. This is the keystone "green out of the box" contract for the phase (D-04). The plan
ships gate config in the generated `pyproject.toml`, adds `comply` targets to the generated
`Makefile`, and fully types + f-string-modernizes all generated project Python (D-05, D-06).

Purpose: D-04/D-05/D-06 for the `project` template — the flagship generated application and
the surface the CI smoke-test (Plan 05) will enforce.
Output: A gate-configured, fully-typed, f-string-clean `project` template that is green out
of the box under its own shipped ruff/mypy/pytest config.
</objective>
