## Shared Patterns

### Typed hook/`load` signature (event-driven)
**Source:** `cement/ext/ext_json.py:5-14`, `cement/ext/ext_alarm.py:11,54`
**Apply to:** `extension/ext_{{ label }}.py`, `todo-tutorial/todo/main.py` `extend_tinydb`
```python
from typing import TYPE_CHECKING
if TYPE_CHECKING:
    from cement import App  # pragma: nocover
def load(app: "App") -> None: ...
def <name>_pre_run_hook(app: "App") -> None: ...
```

### f-string conversion (D-06)
**Source:** `todo-tutorial/todo/main.py` (already done), `todo-tutorial/todo/controllers/base.py:6-9`
**Apply to:** `project/main.py:62,70,79`, `project/controllers/base.py:6-9`, `script/{{ label }}.py:6-8`
Pattern: `'...%s' % x` ŌåÆ `f'...{x}'`; `"""..%s..""" % (a, b)` ŌåÆ `f"""..{a}..{b}.."""`.

### pdm-backend packaging shape (D-02/D-03)
**Source:** `cement/cli/templates/generate/project/pyproject.toml` (canonical), 01.1-01-SUMMARY
**Apply to:** `todo-tutorial/pyproject.toml`
Key invariants: `[build-system] pdm-backend`; `dynamic=["version"]` + `[tool.pdm.version] source="file"`;
`[dependency-groups].dev`; **no `[tool.pdm.build]` block** (would strip templates).

### `-> None` action typing (request-response controllers)
**Source:** cement's own typed source throughout `cement/`
**Apply to:** every `_default`/`command1`/`@ex` action + every `def main()` across all 5 templates.

### Trimmed gate config (D-04)
**Source:** cement root `pyproject.toml` `[tool.ruff]`/`[tool.mypy]`/`[tool.pytest]` (L62-209)
**Apply to:** `project/pyproject.toml` (and by extension `todo-tutorial` if the planner adds gate
config there too). Trim cement-internal ignores (`A001/A002/C901`, per-file-ignores) and reconsider
`T20` (generated code prints intentionally).
