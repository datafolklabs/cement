## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 2 - Missing Critical] Added mypy `disable_error_code = ["attr-defined"]` to the todo pyproject**
- **Found during:** Task 1 (pyproject creation)
- **Issue:** The plan's enumerated mypy knobs omitted `disable_error_code`, but the todo package source (`todo/controllers/base.py`) uses the Cement handler/Meta dynamic-attr pattern (`self.app.args.print_help()`, `self.app.pargs.foo`). mypy would flag these `attr-defined` and `make comply` (a key_links stated goal via the tinydb override) would fail. - **Fix:** Added `disable_error_code = ["attr-defined"]` with an explanatory comment mirroring the project template.
