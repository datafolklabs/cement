## Access mechanism

| Option | Description | Selected |
|--------|-------------|----------|
| Controller property | Read-only property on `ArgparseController` (proposed `self._command_meta`), resolves the running command via `app.pargs.__dispatch__`; returns `None` outside dispatch. Most ergonomic/discoverable, matches `_meta` idiom. | ✓ |
| app.pargs dunder | Set `self.app.pargs.__command_meta__` in `_dispatch()`, mirroring `__dispatch__`. Robust but verbose, dunder less discoverable. | |
| Opt-in signature injection | Inspect signature; pass meta only when the command declares the param. Closest to Tom's ask but dispatch "magic", hard to type-check. | |

**User's choice:** Controller property
**Notes:** Clarified the `CommandMeta` shape before deciding — confirmed it is a
`@dataclass` (7 fields; `help` lives in `parser_options`), `controller` field is
populated by dispatch time, and the meta object is shared at class level. Name
`_command_meta` accepted as proposed (adjustable if a collision is found).
