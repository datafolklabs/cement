## Sampling Rate

- **After every task commit:** Optional `pdm build` smoke (~3s) confirms cement's own packaging is unaffected by template edits. Template files are excluded from ruff/mypy/coverage so no per-file unit tests apply.
- **After commit 3 (new pyproject.toml lands) and after commit 7 (smoke-test fix lands):** Run `PYTHON_VERSIONS=3.10 make cli-smoke-test`. Intermediate states between commit 3 and commit 7 are intentionally inconsistent and not expected to be green.
- **After every plan wave:** Run `PYTHON_VERSIONS=3.10 make cli-smoke-test` once all plan commits have landed.
- **Before `/gsd-verify-work`:** Full suite must be green — `make cli-smoke-test` on the full 3.10–3.14 matrix AND cement's own `make test` + `make comply` (regression check on `cement/ext/ext_generate.py` and core).
- **Max feedback latency:** ~30s (single-version); ~5 min (full matrix).
